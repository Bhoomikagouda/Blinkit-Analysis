📌 Project Overview

This project is a Blinkit Sales Dashboard developed using Power BI to analyze sales performance, outlet establishment trends, item categories, outlet sizes, and customer ratings.

The dashboard provides interactive visual insights that help understand business performance, sales distribution, and outlet analysis across different categories.

🎯 Objectives
Analyze total sales performance
Track number of items sold
Compare average sales and ratings
Understand outlet performance by size and location
Analyze item types and fat content distribution
Visualize sales trends over time
📊 Dashboard Features
Total Sales Overview
Average Sales
Average Rating
Number of Items
Outlet Establishment Trend
Outlet Size Analysis
Outlet Location Analysis
Fat Content Analysis
Item Type Sales Distribution
Outlet Type Comparison Table
Interactive Filters/Slicers

📈 Key Insights
Total sales reached approximately $1.2M.
Medium-sized outlets contributed the highest sales.
Tier 3 locations showed strong sales performance.
Regular fat products generated more sales compared to low-fat products.
Fruits, snacks, and household items were among the top-selling categories.
